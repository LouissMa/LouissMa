# 🚀 Featured Projects


## 🧠 NEXUS (LifeAgent)

**An Autonomous AI Operating System**

NEXUS explores how large language models can evolve into proactive personal agents.

Implemented components include:

- Retrieval-Augmented Generation (RAG)
- Long-term memory architecture
- MCP-based tool calling
- Multi-agent collaboration
- Planning and self-reflection
- Browser and local computer automation


The project focuses on building a modular and extensible architecture for future autonomous AI systems.


🔗 Repository:
https://github.com/mabingnan/NEXUS



---

## ♟️ AlphaZero Gomoku

**Self-play Reinforcement Learning System**

Implemented an AlphaZero-style framework for Gomoku:

- Monte Carlo Tree Search (MCTS)
- Neural network policy/value estimation
- Self-play training
- Game evaluation pipeline


Exploring how agents can learn complex strategies through interaction and feedback.
